Fidelis Utility

Deliverables:

Low resolution designs, copy

Timeline:

2019

Aspen Fidelis Reliability is a tool for simulating conditions in a plant. When I was brought on as the designer for Fidelis, part of my work included coming up with designs for two tools- the Case Runner tool, and the Case Comparison tool.

The Case Runner tool helps users to scale up their simulations quickly. In Fidelis, each forecast of how a plant will run over time is referred to as a "Lifecycle"; the Case Runner was designed to allow users to run multiple lifecycles using multiple copies of the Fidelis application (instances) and multiple threads. The window provides information about each of the fields via info tooltips, and allows users to review the current progress via dynamic progress bars.

The Case Comparison tool, on the other hand, is useful when users want to compare results from different files. At the beginning of the project, the project manager sent me tables and charts that users would want to be able to compare results for. My initial design of a large tree grid was functional, if bland, but I wasn't happy with how much digging users would have to do to get valuable data. I briefly attempted a more visual box-and-whisker plot design before finally asking the correct question- what were users looking for when they compared the data? A variety of reasons, as it turned out. These reasons are reflected in the final design.

Exchange

HTML/JS/Jquery Prototype

Timeline:

2016

When our product line's resource site was redesigned, I was called in to try and demonstrate some slick animations that could improve the site's curb appeal. Working with my manager at the time, we decided on what interactions should be animated and then I set out making it a reality via a rough prototype. With this prototype we were able to determine some flaws with the initial design- it's a bit weird to have all of the search results light up when you move your cursor quickly, so we determined that the final product would have a delay baked in. Almost all of the items in the viewing pane are clickable, so try it out!

Aspen Basic Engineering (ABE)

Deliverables:

Low and high resolution designs, icons, specifications, copy, user testing

Timeline:

2015 - present

Aspen Basic Engineering is a tool that allows users to create documents about equipment. These records can be used as references or as specification documents for new pieces of equipment. In this project, I was tasked with redesigning the entire ABE user interface as a web application. Undoubtedly my largest project ever, the design and development work for the redesign took nearly two years as we hashed out requirements, made mockups, and made changes. Since the redesign's release, I have continued to work with the team on refining the product.
